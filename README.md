# 🏥 E-Pharmacy

Eczane yönetim paneli ve online ilaç mağazası.  
Admin paneli üzerinden ürün, tedarikçi, sipariş ve müşteri yönetimi yapılırken; müşteri arayüzü üzerinden ilaçlar görüntülenip sepete eklenerek sipariş oluşturulabilir.

🔗 **Canlı Backend:** [https://pharmacy-backend-gjah.onrender.com](https://pharmacy-backend-gjah.onrender.com)

---

## 🚀 Özellikler

- **Kimlik Doğrulama:** JWT tabanlı giriş/kayıt, rol bazlı erişim (admin, vendor, customer)
- **Admin Paneli:** Dashboard, ürün/tedarikçi/müşteri/sipariş yönetimi, filtreleme, CRUD işlemleri
- **Müşteri Arayüzü:** Ana sayfa, ilaç kataloğu, ürün detayı, sepet, sipariş oluşturma, eczane listesi
- **Responsive Tasarım:** Mobil, tablet ve masaüstü uyumlu
- **Modern UI:** Material UI (MUI), React Hook Form + Yup validasyon, Framer Motion animasyonları
